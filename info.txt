🟢 1. Core Features (Foundation – MUST HAVE)
    These are non-negotiable.
    User Registration & Login (JWT / Session-based) 
    Add / Edit / Delete expenses 
    Expense fields: 
        Amount
        Category
        Date
        Notes
    Category management (custom categories) 
    View all expenses (list/table view)
    Monthly & daily summaries
    Filter & search:
    By date range
    By category
    By amount


🔵 2. Dashboard & Visualization
    This is where your UI starts impressing.
    Total spending (monthly / yearly)
    Category-wise breakdown
    Recent transactions
    Visual charts:
    Pie chart (category split)
    Bar graph (monthly spending)    
    Line chart (trend over time)

Set monthly budget (overall)
Set category-wise budgets
Track budget usage
Budget alerts:
50%, 80%, 100% usage warnings
Remaining budget display

🔍 Advanced Filtering
